# Notes

This repository contains technical notes, runbooks, configuration references, and learning materials related to systems administration, Linux, networking, cloud infrastructure, and data center operations.

The goal is to maintain a structured knowledge base that documents practical experience, troubleshooting procedures, and reproducible system configurations.

## Structure

### Linux

Operating system installation notes, workstation configuration, software setup, troubleshooting guides, and supporting dotfiles.

* `linux/rocky/`

### Networking

Network engineering notes, routing and switching concepts, lab documentation, and Cisco-related material.

### AWS

Cloud infrastructure notes, deployment procedures, and service-specific documentation.

### Data Center

Operational procedures, infrastructure concepts, and facility-related documentation.

## Repository Conventions

* Markdown is used for documentation.
* Configuration files are stored under `dotfiles/`.
* Helper scripts are stored under `bin/`.
* Sensitive information such as credentials, private keys, and tokens is never committed.
* Notes are intended to be reproducible and reusable whenever possible.

## Purpose

This repository serves as a long-term technical knowledge base and reference library supporting continued learning and professional development in infrastructure, networking, Linux, cloud, and data center technologies.
