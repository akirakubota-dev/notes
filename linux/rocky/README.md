# Rocky Linux 9 Workstation Setup

This directory documents the configuration of my Rocky Linux workstation.

Topics include:

- Initial installation
- GNOME customization
- SSH and Git configuration
- Vim and Tmux setup
- Dropbox integration
- OBS Studio installation
- MPV configuration
- Linux audio troubleshooting

The goal is to maintain a reproducible workstation build process and document solutions to common Linux administration issues.
